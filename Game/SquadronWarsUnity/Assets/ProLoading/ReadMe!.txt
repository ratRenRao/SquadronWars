Thanks for buying Pro Loading.

To start with pro loading:

- Import the "Pro Loading" package your project.
- Place the prefabs "ProLoading" inside his scene.
- Edit its configuration loading effect to your preference.

Tips:
- To call the effect of loading of "Pro Loading" from any script, do it like this:

For Show:

bl_Loading.Loading = true;

For Hide:

bl_Loading.Loading = false;

For customize position and setting use the "bl_Test.cs", and copy in component.
